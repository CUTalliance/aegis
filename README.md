# Aegis Planner

> Lightweight event planning and roster tool for the CUT Alliance (GitHub Pages).

## Overview

Aegis Planner is a compact React + Vite app for planning events, managing members, and exporting reports and maps.

## Features

- Create and manage events
- Manage member rosters and assignments
- Export Excel and PNG reports
- Interactive tactical map with printable exports

## Tech stack

- React 18
- Vite
- Tailwind CSS
- Zustand for state management

## Getting started

Prerequisites: Node.js 16+ and npm or yarn.

Install dependencies:

```bash
npm install
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Lint source files:

```bash
npm run lint
```

## Project structure

- src/ — application source
- src/components/ — UI components
- src/views/ — page views
- src/services/ — export helpers (Excel, PNG, maps)
- src/store/ — Zustand stores for app state
